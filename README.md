## 変更履歴
### [0.0.1] - 2025/11/01
- 環境構築
### [0.0.2] - 2025/11/01
- **変数宣言**
    - `var a; var b;` などのように変数宣言をすることができます。
    - 現在初期化・型指定には対応してません。
    - 代入もできないので変数と言っていいかなぞです。
### [0.0.3] - 2025/11/02
- **初期化付き変数宣言**
    - `var a = 5;` などのように変数を5で初期化することができます。
- **式**
    - `var a = 5 + 2;` などのように式を書いてその結果を代入数することができるようになりました。
    - 四則演算のみ対応しています。
